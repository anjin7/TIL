## EP32 NoSQL이 뭐죠?

NoSQL의 데이터베이스

### 도큐먼트DB

몽고DB

데이터를 JSON 도큐먼트 형태로 저장

대괄호와 중괄호로 구분해서 저장

```
[
	{"id": 1, "name": "슬리퍼", "price": 30000},
    {"id": 2, "name": "바지", "price": 50000},
]
```

데이터마다 구성이 같을 필요없고, 어떠한 모양이든 어떠한 종류이든 저장할 수 있음
